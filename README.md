# Crate xml_dom_xpath

A Rust crate providing an implementation of W3C XML Path Language (XPath), [Version 1.0](https://www.w3.org/TR/xpath-10/)
over the [Document Object Model Core, Level 2](https://www.w3.org/TR/DOM-Level-2-Core).
